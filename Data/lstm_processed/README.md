# Data Folder Guide

This folder is intentionally split into source data, GEE exports, processed
model data, and EDA outputs.

## Main Files for the Current LSTM Pipeline

- `salinity_with_updated_coords.csv`: cleaned 22-station salinity table with
  updated coordinates. This is the salinity source used by the pipeline.
- `gee_data/era5_land_daily_22_salinity_stations_2020_2023.csv`: ERA5-Land
  daily weather export from Google Earth Engine.
- `gee_data/s2_previous_month_station_composites_2020_2023.csv`: Sentinel-2
  previous-month station composite export from Google Earth Engine.
- `processed/lstm_daily_multisource_2020_2023.csv`: final merged daily dataset
  for LSTM feature engineering/window creation.

## Supporting and Audit Files

- `New_Coor.csv`: updated station coordinates used to rebuild the salinity
  station table and GEE scripts.
- `coordinate_update_report.csv`: audit trail for coordinate matching.
- `processed/lstm_data_quality_report.csv`: compact QA report from the merge
  pipeline.
- `processed/lstm_station_summary.csv`: station-level coverage summary from
  the merge pipeline.

## EDA Outputs

Files under `eda/` are generated by `eda_lstm_dataset.py`. They are not extra
raw data; they are small analysis tables and SVG figures for writing the paper:

- `eda/paper_data_summary.md`: plain-language summary of dataset coverage and
  key descriptive statistics.
- `eda/eda_overview_metrics.csv`: overall source and merged-dataset metrics.
- `eda/eda_station_coverage.csv`: station-level salinity and Sentinel-2
  coverage.
- `eda/eda_monthly_coverage.csv`: month-level observation and weather summaries.
- `eda/eda_feature_missingness.csv`: missingness by column.
- `eda/eda_salinity_monthly_stats.csv`: station-month salinity statistics.
- `eda/eda_feature_correlations.csv`: descriptive Pearson correlations with
  observed salinity max.
- `eda/*.svg`: lightweight figures generated from the EDA tables.

## Raw Archive

- `Raw/`: original Excel files from 2020-2025. The current LSTM workflow uses
  the consolidated 2020-2023 salinity file above, not these raw files directly.
